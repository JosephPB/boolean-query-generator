# Boolean Query Generator

Static webpage for helping users generating Boolean queries for searching in the Brandwatch and Meta Content Library social media listening tools based on phrases and keywords.

Inspired by the wonderful (BooleanBuilder)[https://github.com/rlfordon/BooleanBuilder] by Rebecca Fordon.

## Run

1. Clone the repo
2. Open index.html in web browser

## Functionality

Users can add keywords and phrases in topic groups. Groups can be combined with either OR, AND or NOT operators. The topics are grouped with single or double brackets accordingly. The Brandwatch and Meta Content Library use different syntax which the tool adapts to.
